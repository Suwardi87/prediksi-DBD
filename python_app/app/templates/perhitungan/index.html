1Langkah 1: Input Data
Data pasien DBD diambil dari database. Setiap pasien memiliki 3 fitur: X1 = Usia, X2 = Lama Rawat Inap, X3 = Jenis Kelamin. Target (Y) = Tingkat Risiko yang ditentukan dari jumlah kasus bulanan (Rendah ≤8, Sedang 9-15, Tinggi >15).

2Langkah 2: Random Sampling (Bootstrap)
Setiap pohon keputusan dibangun dari bootstrap sample — pengambilan data acak dengan pengembalian (with replacement) dari dataset asli. Ukuran sample = ukuran data asli ( data), tetapi beberapa data bisa terpilih lebih dari sekali dan beberapa data tidak terpilih sama sekali.

Di setiap node, hanya m = fitur yang dipilih acak untuk mencari split terbaik.

3Langkah 3: Pembuatan Pohon Keputusan
Setiap pohon dibangun dengan algoritma ID3 (Iterative Dichotomiser 3) menggunakan Entropy dan Information Gain. Pada setiap node, m fitur dipilih acak, lalu dihitung Entropy dan Information Gain untuk menentukan split terbaik.
Rumus Perhitungan
Entropy:
E(S) = -Σ pᵢ × log₂(pᵢ)
pᵢ = proporsi kelas i dalam dataset
Information Gain:
IG(S,A) = E(S) - Σ (|Sᵥ|/|S|) × E(Sᵥ)
A = atribut, Sᵥ = subset untuk nilai v
Detail Pohon (3 Pohon Pertama)
4Langkah 4: Perulangan Tahap 2 dan 3
Langkah 2 (random sampling) dan Langkah 3 (pembuatan pohon) diulang sebanyak n_estimators = kali, menghasilkan pohon keputusan yang berbeda-beda. Setiap pohon dibangun dari bootstrap sample yang berbeda.

5Langkah 5: Evaluasi Model (MAE, RMSE, R²)
Evaluasi menggunakan Stratified 5-Fold Cross-Validation. Data dibagi 5 bagian, setiap fold bergantian menjadi data test. Metrik dihitung per fold lalu dirata-rata.
Rumus Evaluasi
MAE (Mean Absolute Error):
MAE = (1/n) × Σ|yᵢ - ŷᵢ|
Rata-rata selisih absolut
RMSE (Root Mean Square Error):
RMSE = √[(1/n) × Σ(yᵢ - ŷᵢ)²]
Akar rata-rata error kuadrat
R² (Koefisien Determinasi):
R² = 1 - (SS_res / SS_tot)
SS_res = Σ(yᵢ-ŷᵢ)², SS_tot = Σ(yᵢ-ȳ)²
Detail per Fold
